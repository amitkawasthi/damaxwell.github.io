Getting Started

  • We will use Piazza for course-related discussions outside of class time. You’ll need to sign up to start using the service.

  • You may find links from Prof. Bueler’s Math 615 site to be helpful.

  • Here are the files for a demo Latex solution set.

      * [HWDemo.tex](HWDemo.tex), the main file with the solutions
  * [prob2.pdf](prob2.pdf), a PDF figure that is included in the solutions
  * [homework.cls](homework.cls), a class file that defines the formatting

  • Here is a beginner’s guide to LaTeX. Please read it, even if you have used LaTeX before.

  • In this class, you may wish to use LaTeX to typeset solutions. If you are new to using LaTeX, I can help. Your first step is to install LaTeX; here are some installation guides.

     * [Latex Installation for Windows](LatexInstallationWindows.pdf)
    • Latex Installation for Mac

    I didn’t write a guide for Linux; if you are using Linux and need help just let me know.
